Elf vs Machine

Unofficial Leaderboard for Advent of Code

The algorithm awards the points based on the difficulty of the puzzle and the completion order. I believe it could help mitigate some of the effects of LLM usage, but even if it won't, it's a fun take on the global scores.

For example, if the median time for completing the puzzle was 300 seconds, then the first place takes 300 points, the second place 0.99 * 300 = 297 points and so forth, up to the last place with 0.01 * 300 = 3 points.

Full formula for each star: points = max(1, round((101 - position) / 100 * median_time_in_seconds))
